# MERN Stack Authentication Project

This is a full-stack web application built using the MERN stack (MongoDB, Express, React, Node.js).
The project provides a simple authentication system with login, registration, and password reset features.

---

##  Features

* User Registration
* User Login
* Forgot Password Page
* Reset Password Page
* Dashboard Page
* Responsive UI using Tailwind CSS
* Frontend–Backend Integration using API

---

## 🛠️ Tech Stack

### Frontend

* React (Vite)
* Tailwind CSS
* React Router DOM

### Backend

* Node.js
* Express.js

---



## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```
git clone https://github.com/samarthrc01/mern-stack.git
```

---

### 2️⃣ Install Frontend Dependencies

```
cd frontend
npm install
```

---

### 3️⃣ Install Backend Dependencies

```
cd ../backend
npm install
```

---

## ▶️ Run the Project

### 🔹 Start Backend Server

```
cd backend
node server.js
```

Server runs on:
👉 http://localhost:5000

---

### 🔹 Start Frontend

```
cd frontend
npm run dev
```

Frontend runs on:
👉 http://localhost:5173

---

## API Endpoints

### Auth Routes

* `POST /api/auth/register` → Register a new user
* `POST /api/auth/login` → Login user

---

## Usage

1. Open the application in browser
2. Register a new account
3. Login using same credentials
4. Access the dashboard

---

## Note

* This project currently uses temporary in-memory storage (no database).
* All data will be lost when the server restarts.
* You can integrate MongoDB for permanent storage.
